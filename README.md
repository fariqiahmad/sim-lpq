# SIM-LPQ

## Panduan Instalasi

1. Clone project
```
$ git clone https://github.com/kamalNasution/sim-lpq.git
```
2. Install
```
$ cd sim-lpq
$ composer install
```
3. Buat user & database MySQL
4. Copy `.env.example` jadi `.env`
5. Edit `.env` sesuai user & database yang dibuat

6. Generate key
```
$ php artisan key:generate
```
